<objective>
Establish the strict, versioned evidence-record boundary and deterministic domain answer normalizers that every prepared example and generated prediction will use.
</objective>

<must_haves>
- Four separate frozen records model examples, generations, scores, and failures without a nullable catch-all or volatile scientific fields.
- Canonical JSON codecs reject unknown fields, unsupported schema versions, malformed enum values, and unstable identifiers.
- Pure GSM8K, ProofWriter, and bAbI normalizers preserve raw material and return explicit invalid or ambiguous outcomes instead of guessing.
- Production-shaped synthetic records round-trip without changing the existing tensor smoke-data interface.
</must_haves>
