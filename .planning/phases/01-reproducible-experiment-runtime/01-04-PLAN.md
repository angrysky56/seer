<objective>
Deliver a deterministic, resumable CPU synthetic experiment through the real CLI/runtime artifact contract, then document and verify offline operation from setup through artifact interpretation.
</objective>

<must_haves>
- A tiny synthetic run constructs only fake/local Python objects, trains on CPU, checkpoints, resumes, and emits production-shaped versioned result records.
- Clean and interrupted/resumed runs have identical canonical scientific record hashes even though volatile operational metadata may differ.
- Ordinary rerun of a complete run changes no files, and corrupt evidence fails validation.
- The full test suite and Ruff pass without network or external weights/data.
- Operations documentation covers setup, explicit download, cached preflight, smoke, resume, replacement, stale lock recovery, verification, and artifact meanings.
</must_haves>
